## 医院建筑 AI 提示词生成器网站

### 任务
用户是建筑设计师，有一个医院建筑AI提示词的Excel文件，需要做一个网站，点击对应提示词、风格、类型，自动生成完整AI可用提示语。

### 数据来源
- Excel 文件：`C:/Users/75739/Desktop/医院建筑AI提示词.xlsx`
- 12个维度，172个提示词条目
  1. 建筑主体类型/医疗功能模块 (20项)
  2. 空间类型 (20项)
  3. 建筑风格 (15项)
  4. 外立面/表皮材质 (15项, 含"适用场景"列)
  5. 色彩方案 (10项, 含"心理效应"列)
  6. 光线/时间 (15项, 含"色温参考"列)
  7. 视角/镜头 (15项, 特殊列布局: 等效焦距+效果)
  8. 环境/周边 (15项)
  9. 氛围/情绪 (10项, 含"适用科室"列)
  10. 常用后缀/渲染参数 (15项, 特殊列布局: 完整后缀+适用工具)
  11. 医疗家具/设备 (12项)
  12. 自然元素/景观 (10项)

### 技术方案
- 纯前端实现: HTML + CSS + JavaScript (无框架依赖)
- 数据从 Excel 解析为 data.js (JSON格式)
- 三个文件: index.html, style.css, app.js, data.js
- 项目路径: G:\SYF_Project\提示词网站

### 功能特点
- 12个维度的标签式选择器，支持多选
- 中英文/双语三种显示模式切换
- 实时生成逗号分隔的AI提示词
- 维度10(渲染参数)为单选模式(选一个渲染风格)
- SD负面提示词自动分离显示
- 搜索功能(中英文模糊搜索)
- 一键复制提示词/负面词
- 维度折叠展开
- 响应式设计，支持移动端
